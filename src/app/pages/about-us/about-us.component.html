Chic-Shop is a fictional e-commerce platform designed and developed as part of my professional portfolio to demonstrate frontend architecture, component-based UI design, and state management in a modern Angular application.

This project was built using:

  • Angular 17 with standalone components
  • TypeScript for strong typing and scalable logic
  • HTML & SCSS for responsive and structured layout
  • Bootstrap 5 for consistent, mobile-first design
  • Custom services and models to simulate data handling
  • GitHub Pages as the deployment platform
